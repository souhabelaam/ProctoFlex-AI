# ProctoFlex AI

ProctoFlex AI is an AI-powered online exam proctoring platform with a FastAPI backend, React web dashboard, and Electron desktop client.

## Tech Stack

- Backend: FastAPI, PostgreSQL, Redis
- Frontend: React, Vite, TypeScript
- Desktop: Electron
- Infra: Docker Compose

## Features

- Real-time exam session monitoring
- AI-assisted suspicious activity detection
- User/session management dashboard
- Web and desktop client support

## Quick Start

```bash
# Clone and run with Docker
git clone https://github.com/souhabelaam/ProctoFlex-AI.git
cd ProctoFlex-AI
docker compose up -d --build
